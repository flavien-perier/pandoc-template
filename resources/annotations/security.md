[^SSO]: Un SSO (Single Sign-On) est un proxy d'authentification unique permettant de se connecter sur plusieurs applications avec les mêmes identifiants.
[^SPOF]: SPOF (Single Point Of Failure) ou "Point de défaillance unique" désigne un composant central d'une architecture dont tous les autres composant dépendent.
[^JWT]: JWT (JSON Web Token) est une méthode d'authentification sans état basé sur un système cryptographique.
